数据恢复

一个文件系统被格式化的瞬间会分为两部分，inode（元数据，文件的大小，时间，指针等等）和block（文件系统最小存储单元，一个文件至少占用一个block）

格式化的时候只是把元数据删除了，但数据块没有删除，下次有数据写进来的时候会将数据覆盖 像以上情况数据块被覆盖后就很难恢复了

压缩解压缩

gzip l 压缩后格式为：*.gz l 这种压缩方式不能保存原文件；并且不能压缩目录 l 举例：

bzip2 l 压缩后格式为：.bz2 l 参数 -k ：产生压缩文件后保留源文件 l 示例

zip l 相比于gzip：可以压缩目录；可以保留源文件 l 选项： -r ：递归压缩所有文件和目录 l 示例

tar 命令参数

  1. -z ：用gzip来压缩/解压文件

  2. -j：  用bzip2来压缩/解压文件

  3. -v： 显示tar处理文件的信息

  4. -c： 创建新的档案文件

  5. -x： 解压缩文件或目录

  6. -f：  使用档案文件或者设备
